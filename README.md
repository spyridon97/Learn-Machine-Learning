# ML-Algorithms
This repository includes a collection of algorithms for Machine Learning.

## Algorithms
*   Interpolation:
    *   Lagrange
    *   Natural Cubic Splines
*   Optimization:
    *   Least Squares
*   Linear Models
    *   Linear Regression
    *   Logistic Regression
*   Clustering:
    *   KMeans
    *   Hierarchical Agglomerative Clustering
*   Trees:
    *   Decision Tree 
*   Ensemble: 
    *   AdaBoost
