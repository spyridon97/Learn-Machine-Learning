# Algorithms

This package includes:
*   Interpolation:
    *   Lagrange
    *   Natural Cubic Splines
*   Optimization:
    *   Least Squares
*   Linear Models
    *   Linear Regression
    *   Logistic Regression
*   Cluster:
    *   KMeans
    *   Hierarchical Agglomerative Clustering
*   Trees:
    *   Decision Tree 
*   Ensemble: 
    *   AdaBoost
